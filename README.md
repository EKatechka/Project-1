Парсер сайта торгов

Что делает программа?

Программа помогает:
- Смотреть лоты - показывает все доступные товары
- Сортировать - от самых дорогих к самым дешевым  
- Фильтровать - искать лоты по цене
- Сохранять - сохраняет результаты в файлы
- Статистика - показывает интересные числа

  Откуда данные?

В этой версии программа использует **демонстрационные данные**, потому что реальные сайты часто защищены.

**Настоящий источник:** https://torgi.org/index.php?class=Auction&action=List&mod=Open&AuctionType=All

Установка (шаг за шагом):

1. Скачайте Python
- Зайдите на https://python.org/
- Скачайте и установите Python 3.8+
- При установке отметьте "Add Python to PATH"

2. Скачайте файлы программы
- Нажмите зеленую кнопку "Code" на GitHub
- Выберите "Download ZIP"
- Распакуйте архив в любую папку

3. Установите зависимости
Откройте командную строку (Terminal) и введите:

```bash
pip install -r requirements.txt
